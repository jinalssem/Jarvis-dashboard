# 🤖 Jarvis Trading Dashboard

## ✨ Features

- 🎨 실시간 워드 클라우드 (거래 시각화)

## 🚀 Quick Start

### 로컬 실행
```bash
# 브라우저에서 index.html 열기

```

```

3. 완료! 즉시 URL 생성됨

vercel
```
